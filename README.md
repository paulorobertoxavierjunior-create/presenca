# Presença

Presença é um espaço de treino humano assistido por tecnologia.

A ferramenta foi criada para ajudar o usuário a observar sua própria expressão, fortalecer clareza, ritmo, firmeza e consciência durante a fala, e se preparar para interações mais avançadas dentro do ecossistema Elayon.

## Proposta

O Presença não é apenas um medidor de sinais.  
Ele é uma ferramenta de:

- autoconhecimento
- treino de expressão
- observação da fala
- fortalecimento da presença
- preparação para uso consciente de tecnologia

## Fluxo do app

index.html

## Modos previstos

- fala livre
- treino de apresentação
- simulação de aula
- simulação de diálogo
- análise de áudio

## Núcleo

O sistema combina:

- CRS (Captação de Ritmos e Sinais)
- leitura orientadora com filtro Elayon
- progressão por fases

## Objetivo

Ajudar o ser humano a se ver no processo, crescer em clareza e usar tecnologia com mais consciência, firmeza e responsabilidade.
