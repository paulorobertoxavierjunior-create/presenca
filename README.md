# Presença

Aplicativo web para acompanhar métricas de presença durante a fala.

## Como abrir
- GitHub Pages: Settings → Pages → Deploy from branch → `main` → `/ (root)` → Save

## Arquivos
- `index.html`
- `styles.css`
- `app.js`
- `IMG-20260218-WA0015.jpg`

## Observações
- O microfone depende de permissão do navegador.
- Funciona em HTTPS (GitHub Pages).
