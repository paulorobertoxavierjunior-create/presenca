Integração de Sistema
Entrada — conecte a voz do seu sistema aqui

Este módulo recebe áudio via microfone local. Para integrar com um chatbot ou sistema externo, direcione o fluxo de entrada de voz para este endpoint antes de qualquer resposta da IA receptora. O CRS lê o estado do emissor e emite o vetor de calibração.

POST https://nucleo-crs-elayon.onrender.com/api/crs/analisar Authorization: Bearer {token_supabase} Content-Type: application/json // Payload: vetor extraído localmente pelo módulo Presença // Campos: duration_sec, silence_pct, pause_count, // mean_pause_ms, oscillation_pct, continuity_pct, // energy_pct, spectrum_snapshot, context
Saída — onde sua IA recebe o relatório

O núcleo CRS devolve um relatório estruturado com o padrão identificado e a instrução de calibração de interação. Configure seu backend para consumir este JSON antes de gerar qualquer resposta ao usuário.

{ "ok": true, "calibracao_ref": { "timestamp": "2026-04-28T22:16:37Z", "vetor": { ...métricas da calibração... } }, "leitura_sessao": { ...métricas da sessão atual... }, "analise_sugestiva": "Fluxo Moderado", "sugestao_ia": "Manter tom claro com validações breves entre blocos de conteúdo." }
